# Support

For installation and usage help, start with the [documentation](README.md#documentation) and run `cairn doctor --json`.

- Report reproducible defects with the bug-report issue template.
- Request adapters with the adapter-request issue template.
- Report security vulnerabilities privately as described in [SECURITY.md](SECURITY.md).
- Please do not use public issues for security reports or personal support requests containing private logs.
