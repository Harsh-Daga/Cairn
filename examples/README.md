# Cairn examples

Small, CI-checked workflows for the current local product (not the retired pipeline runner).

| Example | Teaches |
|---------|---------|
| [e2e-demo](e2e-demo/) | Deterministic demo workspace → receipt → check |
| [ci-gate](ci-gate/) | `cairn check` in GitHub Actions |
| [otlp-ingest](otlp-ingest/) | `POST /v1/traces` with a sample OTLP/JSON body |
| [mcp-setup](mcp-setup/) | `cairn mcp install` for agent clients |
| [export-archive](export-archive/) | Scrubbed static export + portable archive |
