# FIN-01 Release gate acceptance

Date: 2026-07-19

## Single local command

```bash
uv run python scripts/release_check.py
```

Runs, in order:

1. Ruff lint + format check  
2. mypy `--strict server`  
3. pytest  
4. UI lint / Prettier / typecheck / Vitest / OpenAPI drift  
5. UI build + gzip bundle budgets  
6. Playwright e2e (Chromium + Firefox/WebKit `@cross-browser`)  
7. Version / changelog / README / GitHub YAML / generated artifact drift  
8. Offline static asset check + publish workflow invariants  
9. Wheel/sdist contents + clean-wheel import/CLI/doctor  
10. Two-build reproducibility assessment  

## Flags

| Flag | Use |
|------|-----|
| (default) | Full local/release-PR gate |
| `--packaging-only` | CI packaging job / publish prep (no lint/test/e2e matrix) |
| `--skip-e2e` | Local iteration without Playwright |
| `--skip-reproducibility` | Skip dual rebuild compare |

CI/publish workflows call `--packaging-only` because the quality matrix already runs in dedicated jobs.
